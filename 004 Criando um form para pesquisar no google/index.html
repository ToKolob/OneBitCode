Criando um form para pesquisar no Google

Prática!

Explicação

Aqui usamos a label form para abrir o formulário e dentro do atributo action informamos para qual servidor será enviado o formulario.

A tag lable informa o usuario o que se deseja o párametro for vai determinar para qual input ela é atribuida. Em seguida usamos o input do tipo texto e colocamos o atributo placeholder para deixar um valor exemplo dentro da caixa de texto. Sem esquecer de informar o id igual o informado no label.

E incluimos um input do tipo submit para enviar os dados.

Ainda existe uma infinidade de atributos para se explorar no w3schools.

Code

      <form action="https://www.google.com/search">
        <label for="q">Pesquisar no google</label>
        <input type="text" id="q" name="q" placeholder="qual quer coisa" required>
        <input type="submit" value="Pesquisar">
      </form>
