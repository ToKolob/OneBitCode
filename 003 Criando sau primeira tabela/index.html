Exercício: Criando sua primeira Tabela

Referência com base em w3schools

Para criar a tabela, vamos usar as tags Table, tr, th, td.

Respectivamente para inserir a tabela, adicionar um table row, table head e table data.

E para melhor organizar, vamos usar caption, thead e tbody, respectivamente para colocar o titulo, e separar o cabeçalho do corpo da tabela.

E para finalizar e deixar pareendo uma tabela, vamos adicionar a tag table o atributo border='1'.

Pratica!

Massas do Sistema Solar
Componente Massa

Júpiter 0,10%
Demais Planetas 0,04%
Cometas 0,01% (?)
